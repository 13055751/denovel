# 怎么用？
**注意** 有部分网站有CC保护，请合理安排间隔时间 "-s"
## 终端（不推荐）
虽然功能更全，但是是单任务的。
```sh
    deno run -A main.ts
```
强烈建议：搭配以下flag使用:
 - `-e` 输出epub
 - `-s` 请求间隔时间，重试不受影响
 - `-p` 有些奇葩的网站内容自带章节标题且内容零散，指定可以避免程序自动添加网页给的标题

## 网页端
![预览](image.png)
```sh
    deno run -A server.ts
```
打开"127.0.0.1:8000"即可

## 转换epub(支持传入文件夹批量转换)
```sh
    deno run -A 2epub.ts <文件(夹)>
```
